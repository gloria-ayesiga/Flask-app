Assignment Overview

As an R and Python developer working for a data analysis company in Washington, DC, you have been assigned the task of developing a survey tool for collecting participants' data. The objective is to analyze their income spending in preparation for a new product launch in the healthcare industry.

Update assignment overview.

Assignment Description

In this final project assignment, you are required to:

  1. Web Development with Flask:

    • Create a simple webpage using Flask to collect user data.

  2. Data Storage with MongoDB:

    • Store user details, including Age, Gender, Total Income, and Expenses, in MongoDB.
    • Use checkboxes for expense categories (utilities, entertainment, school fees, shopping, healthcare), each with a corresponding textbox to insert amounts spent.

  3. Data Processing with Python or R:

    • Create a Python or R class named "User."
    • Loop through the collected data and store it in a CSV file.
    • Load the CSV file into a Jupyter notebook.

  4. Data Visualization:

    • Perform the following visualizations in your Jupyter notebook:
      • Show the ages with the highest income.
      • Show the gender distribution across spending categories.
      • Export the charts for use in a PowerPoint presentation for client use.

  5. Deployment on AWS:

    • Host the Flask application on Amazon Web Services (AWS).

Note: Submit your work as a zipped file or provide a link to your GitHub repository, including a README file with instructions. 

Update assignment description.

Assignment Instructions

To successfully complete this final project, follow these steps:

  1. Create a Flask web application for data collection.

  2. Implement MongoDB to store user data, including the specified details and expense categories.

  3. Develop a Python or R class named "User" for data processing.

  4. Loop through collected data and store it in a CSV file.

  5. Load the CSV file into a Jupyter notebook for data analysis.

  6. Perform visualizations as described, and export the charts.

  7. Host your Flask application on AWS.

  8. Submit your work as a zipped file or provide a link to your GitHub repository, including a README file with detailed instructions.
